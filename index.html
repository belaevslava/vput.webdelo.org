Notification Center

Push notification

  • Позволяет получать уведомления на всех типах устройств
  • Позволяет снизить время реакции на события в системе
  • Нет необходимости разрабатывать отдельное веб-приложение

SMS notification

  • Мультиязычная отправка СМС оповещений в зависимости от языка аккаунта.
  • Возможностью подключать различные движка для отправки СМС.
  • Возможность отпарвлять СМС используя разные сервисы для разных ЗОН получателей. Экономит деньги.

E-mail notification

  • Мультиязычные шаблоны для рассылки по языку аккаунта
  • HTML вёрстка шаблонов
  • Адаптированные шаблоны под устройства
